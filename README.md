# Bootstrap Authentication UI

This project is a simple multi-page authentication interface built using **HTML, Bootstrap 5, and Custom CSS**.

## Pages Included
- **Login Page**
- **Registration Page**
- **Forgot Password Page**
- **Reset Password Page**
- **Dashboard Page**

## Features
- Bootstrap 5 responsive design
- Card-based form layouts
- Styled form controls and buttons
- Password visibility toggle
- Navigation bar on dashboard
- Custom professional color scheme
- Google Fonts integration
- Hover effects and smooth transitions
- Box shadows and modern spacing
- Background gradient styling

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
